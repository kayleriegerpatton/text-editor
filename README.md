## Description

This single page application uses IndexedDB and Express to allow users to edit text in their browsers as well as offline when installed.

### Technologies

- CSS
- Express.js
- JavaScript
- IndexedDB

## Getting Started

### Installation

```
git clone https://github.com/kayleriegerpatton/text-editor.git
cd text-editor
npm i
```

### Run the App

```
npm run start
```
